About This Project

This project analyzes product placement performance using Tableau dashboards and stories. It helps understand product positioning, pricing, promotions, and customer buying behavior through interactive visualizations.

  • Analyze product categories and pricing trends.
  • Understand customer demographics and sales performance.
  • Create interactive Tableau dashboards and stories.

Using Tableau Public, this project explores retail product placement by analyzing sales trends, customer demographics, seasonal demand, pricing strategies, and promotional performance. The dashboard enables business users to discover actionable insights that improve product visibility and increase sales.
